# Admin UI Scaffold

This folder is a static scaffold for the dedicated PubAuth admin UI surface.

Included pages:

- Dashboard
- Products
- Workspaces
- OIDC Clients
- Gateway Policies
- Roles & Assignments
- Audit Logs
- Provider Setup

Current scope:

- Typed navigation and page contracts
- API client contract ready for `/api/*` wiring
- Shell/page metadata for a future React/Vite surface

Not yet wired:

- Runtime bundler
- Rendered components
- Data fetching hooks
- Authentication/session handling for the standalone admin surface
